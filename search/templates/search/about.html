Hilfe und Information

Suche

Über die Suchzeile lässt sich nach Städten, Stadtteilen und Postleitzahlen in NRW suchen. Nach dem Abschicken der Suche über den „Suchen“ Button oder die Enter-Taste, wird der eingegebene Ort in der Karte angezeigt und rot markiert. Dies kann einige Sekunden dauern. Möchte man auf die Stadtteile einer Stadt zugreifen, kann man dies über verschiedene Wege. Der Stadtteil (Bsp. Chorweiler) kann direkt über die Suchzeile eingegeben werden und ist so direkt in der Karte sichtbar. Alternativ kann das Dropdown-Menü, in welchem die aktuelle Suche steht, geöffnet werden. Die nun sichtbare Auswahl zeigt alle Stadtteile der aktuellen Stadt an (falls vorhanden). Bei einem Klick auf eine Auswahl wird so eine neue Suche getätigt. Die letzte Möglichkeit den gesuchten Ort zu spezifizieren, ist über den Klick auf einen Stadtteil direkt auf der Karte. Beim Herüberfahren über die Karte mit dem Mauszeiger wird der Name des aktuellen Ortes angezeigt. Mit einem Doppelklick wird eine Ebene in den Ort hineingezoomt.

 

Filter

Nachdem eine erfolgreiche Suche durchgeführt wurde, werden weitere Elemente auf der Webseite sichtbar. Direkt unter der Suchzeile gibt es eine Reihe von thematisch geordneten Dropdown-Menüs, welche Filter enthalten, die auf der Karte angezeigt werden können. Mittels eines Klicks auf den entsprechenden Radio-Button wird ein Filter ausgewählt. Nun kann bei Bedarf noch die minimale und maximale Entfernung gewählt werden, die das gewählte Objekt (Filterkategorie) vom gewünschten Wohnort entfernt sein soll. Diese Werte können jedoch auch frei gelassen werden (beide oder nur einer von beiden), um nur die Filter ohne eine Reichweite anzuzeigen. Über den „Hinzufügen“ Button wird ein Filter der aktuellen Auswahl hinzugefügt, jedoch noch nicht angewandt. Hat der Nutzer alle gewünschten Filter ausgewählt und hinzugefügt, kann über den „Filter anzeigen“ Button die Filterauswahl auf der Karte angezeigt werden. Sollen nun noch die eingetragenen Radien mit einbezogen werden, kann dies über den „Schnittmenge berechnen“ Button geschehen. Hier ist zu beachten, dass bei einer zu großen Menge an ausgewählten Filtern möglicherweise keine Überschneidungen mehr gefunden werden oder die Karte durch zu viele angezeigte Filter zu überladen wirkt und sich einzelne Marker überlagern. Wir empfehlen daher nicht mehr als drei Filter auszuwählen.

Um einen Filter aus der aktuellen Auswahl zu entfernen, muss dieser einfach nur angeklickt werden (Animation: Name des Filters wird durchgestrichen) und verschwindet danach.

 

OpenData

Die OpenData sind bisher nur für die Stadt Köln implementiert. Um sich die Daten anzeigen zu lassen, gibt es erneut mehrere Optionen. Zunächst muss der Schieberegler „OpenData“ aktiviert werden. Danach kann sich der Nutzer, wenn er sich auf Stadtbezirk- oder Stadtteilebene befindet, mittels Hover auf den Stadtteil alle hinterlegten OpenData anzeigen lassen. Diese erscheinen in Form einer kleinen Sprechblase direkt auf der Karte. Möchte man speziellere Daten, wie beispielsweise die LKW-Verbotszonen anzeigen lassen, kann man diese direkt über den Schieberegler aktivieren. Die Verbotszonen werden in Form einer farbigen Flächer direkt auf der Karte eingezeichnet. Bei erneutem Betätigen des Reglers werden die Daten wieder ausgeblendet.

Hat man alle Filter in der Karte anzeigen lassen und befindet sich im richtigen Stadtteil, empfiehlt es sich zur genaueren Beurteilung des gefundenen Gebietes die farbige Markierung auf der Karte auszuschalten. Dafür muss einfach nur der „Transparenz“ Schieberegler betätigt werden und die farbige Markierung verschwindet. Um trotzdem weiter den gewählten Stadtteil im Auge zu behalten, bleibt dessen Umrandung weiterhin bestehen.

 

Die Karte

Auf der Karte werden sowohl der gesuchte Ort als auch die ausgewählten Filter und OpenData angezeigt. Möchte man von der Satellitenansicht auf die gezeichnete Karte wechseln, kann dies über das Symbol in der oberen rechten Ecke geschehen. Gezoomt kann mit Doppelklick oder mit den Plus / Minus Buttons direkt auf der Karte: Der aktuelle Maßstab wird dabei am unteren Kartenrand angezeigt. Durch Festhalten und Verschieben des Kartenausschnitts kann sich auf der Karte bewegt werden. Bei einer Suche wird der Nutzer jedoch direkt auf den richtigen Kartenausschnitt weitergeleitet.
